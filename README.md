# web_project_homeland
Sprint 5 - De patria a patria
Se creó una página en la cual trata de las diferentes ciudades de los colegas de TripleTen, la finalidad es crear una página que sea responsiva para que sea posible poder observar este contenido en diferentes pantallas como computadoras, tablets o celulares con diferentes tamaños.
Se utilizó la propiedad de media queries la cual nos brinda la fácilidad de generar los estilos en diferentes proporciones.
